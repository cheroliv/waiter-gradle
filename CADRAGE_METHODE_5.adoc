== Story Mapping (priorise)

=== EPIC 0 — Onboarding

[cols="1,3,1,1", options="header"]
|===
| Prio | User Story | Persona | Valeur
| P0 | En tant que developpeur, je veux `git clone engine && ./gradlew tasks` afin d'avoir la liste des taches disponibles. | Tous | Critique
| P0 | En tant que developpeur, je veux Docker Compose avec l'engine monte en volume afin d'avoir un environnement pret a l'emploi. | Tous | Critique
| P1 | En tant que developpeur, je veux un README decrivant les prerequis et les commandes. | Curieux YouTube, Apprenant | Haute

[cols="1,3,1,1", options="header"]
|===
| Prio | User Story | Persona | Valeur
| P0 | En tant qu'utilisateur, je veux `./gradlew aggregateGraphs` afin d'obtenir le diagramme du DAG foundry. | Createur, Dogfooder | Critique
| P0 | En tant que createur, je veux `./gradlew decomposeIntention -Pintention="..."` afin d'obtenir un plan structure [EPIC][US][TASK]. | Createur | Critique
| P1 | En tant qu'utilisateur, je veux `./gradlew {tache} --mode=ci` afin de deporter l'execution sur GitHub Actions. | Tous | Haute
| P1 | En tant qu'utilisateur, je veux un message d'erreur lisible si une tache echoue. | Tous | Haute
| P2 | En tant qu'utilisateur, je veux un timeout configurable par tache. | Tous | Moyenne
|===
