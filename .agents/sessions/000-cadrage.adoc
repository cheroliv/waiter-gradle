== Decisions Cle

. *Tool-agnostique* : interfaces shell (`./gradlew`) + protocole Git (commit+push). Zero adherence a open-webui, opencode ou tout outil specifique.
. *Plugin engine* : `com.cheroliv.waiter` dans engine N3. Point d'entree unique.
. *Protocole Git* : commit+push comme canal de communication herite de `codex-gradle/AUGMENTED_CONTEXT_PIPELINE.adoc`
. *Experience de creation de contenu complete* : intention → planification → generation → publication. Tout depuis le terminal.
. *6 EPICs, 23 US* estimees et priorisees, Sprint 1 atomise en 23 taches.
. *Open source Apache 2.0*, public sur `github.com/cheroliv/engine`
